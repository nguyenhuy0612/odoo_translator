How It Works

Get started in 3 simple steps

1

Import Files

Select one or more .po files from your Odoo module's i18n folder. Files are automatically merged and deduplicated.

2

Configure & Translate

Set source and target languages. Click "Translate All" or select specific entries. Works online or offline!

3

Export & Use

Export translated .po file (and optional .mo). Copy back to your Odoo module and update!

Frequently Asked Questions

Everything you need to know

Is PO Translator really free?
Yes! PO Translator is 100% free and open source under MIT license. You can use it for personal or commercial projects without any cost.
Do I need an API key?
For online AI translation, you need a free Google Gemini API key from Google AI Studio. For offline mode with built-in glossaries, no API key is needed!
Which languages are supported?
Currently supports: English, French, Spanish, German, Italian, Portuguese, Dutch, and Arabic. Offline mode works for EN‚ÜîFR and EN‚ÜîES.
Can I use it offline?
Yes! Enable "Offline mode" in the sidebar to use built-in glossaries for English‚ÜîFrench and English‚ÜîSpanish translations without internet.
How accurate are the translations?
AI translations are ~95% accurate for Odoo-specific terminology. The tool uses carefully crafted prompts with domain glossaries. Always review translations before deploying to production.
Does it work with Python 3.13?
Requires Python 3.11+. Some dependencies (fasttext-wheel) don't support Python 3.13 yet. We recommend Python 3.11 or 3.12.
Can I contribute?
Absolutely! Contributions are welcome. Check out the GitHub repository for issues, feature requests, and contribution guidelines.
